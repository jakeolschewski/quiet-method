# Quiet Method — Icon Asset Descriptions

All icons should be created as PNG files at the specified sizes.

## Design Direction
- **Style:** Minimal, geometric shield shape with a subtle inner calm/wave motif
- **Palette:** Indigo (#6366f1) shield with a soft white interior check or wave
- **Background:** Transparent
- **Feel:** Protective but gentle — not aggressive security, more "quiet confidence"
